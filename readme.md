# Ajaro - Workshop PoS

## Requirements

* Git
* PHP >= 7.1.3
* Composer
* PostgreSQL 10
* [Laravel](https://laravel.com/docs/5.8/)
* DBeaver

## How to run

* clone this repo
* execute `composer install`
* create and edit file `.env`
* execute `php artisan migrate`
* start server by execute `php artisan serve`
